Code Island Projects

Code Island has recently completed its first project and has several more in development. All of our work is open source and available on Github.

Providence Mayoral Candidates Open Data Questionnaire—We asked the three candidates to go on the record regarding open data...and all three responded.
STATUS: Complete

Rhode Island Community Wiki—Our ongoing, longterm wiki project lets anybody post information about anything and everything Rhode Island. Help us make it the best RI website EVAH!
STATUS: Ongoing

RI Budget Visualization—Using five years of check-register data recently released on the RI Transparency Portal, we are creating a visualization tool that will let you drill deep into the state's spending patterns.
STATUS: In development

RIPTA App—RIPTA has released their schedule data in an open standards format, and the Transit app now supports RIPTA. We are working with RIPTA to get their live GPS data standardized and released so that we can make a state-of-the-art, real-time tracker.
STATUS: Awaiting dataset
